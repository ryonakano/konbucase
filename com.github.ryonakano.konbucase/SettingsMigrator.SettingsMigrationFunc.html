private delegate bool SettingsMigrationFunc (Settings settings, Variant old_val)

Migrates an app preference in an old key in settings.

Parameters:

settings

the target GLib.Settings for migration. The preference should be saved to new key(s) using setters of GLib.Settings e.g. GLib.Settings.set_value

old_val

an app preference that the old key has

Returns:

true if succeeds, false otherwise
